Dysplay and positioning

property tag description
z-index property position: absolute
z-index +1
 specifies how far back or how far forward an element will appear on a web page when it overlaps other elements.
position position: static , relative , absolute or fixed static= default/normal position
relative= relative to lenght you set (right: 20px)
absolute= is positioned relative to the nearest positioned ancestor and it will be removed from the normal flow.
fixed= its stays at the same position
clear property clear: left,right This determines that no other elements within the same containing element are allowed to float on the left side of this element
dysplay property display: inline, blocks, inline blocks block elements take up the full width of their container with line breaks before and after, and can have their height and width manually adjusted.
Inline elements take up as little space as possible, flow horizontally, and cannot have their width or height manually adjusted.
Inline-block elements can appear next to each other, and can have their width and height manually adjusted
float position float: left, right, top, bottom (can be mixed) float property determines how far left or how far right an element should float within its parent element.
